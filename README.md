# Simple data Analysis using python
## *Five questions*
1. Which age group has the highest average blood glucose level?
  
   Objective: Identify the most at-risk age category.
   
   **Python skills:**
      - Grouping 
      - Aggregation
2. Is there a relationship between BMI and Blood Glucose?

   Objective: Determine whether higher BMI is associated with higher blood sugar levels.

   **Python skills:**
      - Correlation analysis
      - Scatter plots
      - Visualization with Matplotlib/Seaborn
3. Which treatment outcomes are most common?
  
   Objective: Understand patient recovery trends.
  
    **Python skills:**
    - Frequency analysis
    - Bar charts
4. Do smokers have higher blood pressure than non-smokers?

    Objective: Compare health indicators between smoking groups.

    **Python skills:**
      - Filtering data
      - Group comparisons
      - Further analysis:
      - T-test using scipy.stats
5. Which facility handles the most diabetes patients and what are their outcomes?

   Objective: Evaluate workload and performance by facility.

    **Python skills:**
    - Grouping
    -	Pivot tables
